# OpenCode 工作区

本仓库是 OpenCode 会话暂存区与能力增强工作区。

## 核心用途

1. **会话暂存（Staging）**：不确定归属的 OpenCode 会话先放在这里处理、整理或等待归档。
2. **技能适配（Skill Adaptation）**：将 Codex 生态中的 skills、hooks、工作流和最佳实践适配到 OpenCode 环境。
3. **能力增强（Enhancement）**：探索、实验和沉淀能提升 OpenCode 使用效率的规则、脚本、prompt 模板和自动化流程。

## 当前入口

- `AGENTS.md`：项目宪法与工作规则。
- `docs/PROJECT_STATUS.md`：当前状态、风险和待办的唯一入口。

## 目录说明

- `docs/`：项目文档、迁移指南和规则草案。
- `sessions/active/`：当前活跃、尚未归档的暂存会话。
- `sessions/archive/`：已归档或已迁移到其他项目的会话。
- `skills/`：本地技能文件、补丁和适配脚本。
- `scripts/`：本项目使用的自动化脚本。
- `.ai_state/`：adaptive-intent-loop 任务状态。
